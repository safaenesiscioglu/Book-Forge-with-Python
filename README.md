# Book-Forge
Book Forge is a pure Python web app that converts PDFs to A5 format. Two modes: Preserve keeps the original layout, cover, and images while adding an invisible OCR text layer for searchability. Reflow extracts text via OCR and rebuilds it as a clean EPUB or A5 PDF. Built with PyMuPDF, pytesseract, ebooklib, and ReportLab.
